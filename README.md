# MaeriWeb

MaeriWeb is the official web application for **Maeri**, an original tabletop role-playing game system.

The goal of this project is to provide players with a fast, accessible, and mobile-friendly tool to play Maeri using only their smartphones during in-person sessions.

This repository contains the foundations of the Maeri Web App, including core rules, gameplay tools, and future narrative content.
